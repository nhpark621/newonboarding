# 체험형 경쟁사 분석 대시보드 – 온보딩 프로세스 데모

본 프로젝트는 신규 잠재 고객이 자사 서비스를 **간단히 체험**할 수 있도록 설계된 체험형 웹 온보딩 프로세스입니다. 사용자는 ‘경쟁사 고민 입력 → 맞춤형 서비스 추천 → 회원가입 → 대시보드 체험’의 간단한 흐름을 통해 자사의 데이터 분석 서비스를 경험할 수 있습니다.

---

## 🔍 프로젝트 개요

- **목표**: 서비스 전환율을 높이기 위한 인터랙티브 체험형 웹 온보딩 프로세스 구현
- **대상 사용자**: 자사 서비스에 관심은 있지만 아직 이용하지 않은 신규 잠재 고객
- **접근 방식**: 간단한 입력과 선택만으로도 실감나는 대시보드 체험 가능
- **활용 목적**: Lean MVP를 넘어, 실사용자 반응을 수집하는 Superior MLP 테스트

---

## ✨ 주요 기능

1. **경쟁사 고민 입력**
   - 자유 입력 또는 추천 태그 선택을 통해 사용자의 니즈 파악
2. **맞춤형 서비스 추천**
   - OpenAI API를 통해 사용자의 고민과 산업군을 분석하고, 최적의 분석 항목 3~4개 제안
3. **서비스 카드 선택**
   - Hover 미리보기 및 간단한 설명 제공, 사용자는 원하는 카드 선택 가능
4. **회원가입 및 추가 정보 입력**
   - 이메일, 비밀번호, 회사명, 소속 팀, 담당 브랜드 등 최소 정보 수집
5. **체험형 대시보드 이동**
   - 입력된 고민에 맞춰 개인화된 경쟁사 분석 체험 제공
